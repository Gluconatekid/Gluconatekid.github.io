Software Development


Welcome to my software projects!

All of the projects on this website are my own work, including the images. On the navigation bar above, you can see the projects are organized into three categories: graphical user interface (GUI), console output, and web. Both the GUI and console projects were written and run using the Netbeans IDE. The text editor I used for this website, the quiz and the art gallery was Notepad++. By hovering the navigation bar above, you can access pages to see explanations and code for the following projects:

  • A GUI Java application for to do lists and goal setting (written in Java & JavaFX)
  • A quiz on the American Revolution that calculates your score (written in HTML5, CSS & Javacript)
  • An art gallery displaying some of my art (written in HTML5 and CSS)
  • A console output program that calculates your shopping bill (written in Java)
  • A console output program that estimates the cost of building roads and bridges (written in C++)

Obviously, putting this website together was another project; bigger than I'd expected. This page was written in HTML5 and CSS. I published this website using GitHub.
